This To-Do application was developed using Angular, giving me the opportunity to learn and practice essential concepts of this framework. Throughout the development process, I explored and deepened my understanding of:

Angular project structure – organizing components, modules, and services. Data Binding and Directives – using [(ngModel)], *ngFor, and *ngIf to manage and display dynamic data. Services and Dependency Injection – creating a service to handle tasks and share data between components. Local Storage – saving and persisting tasks for a better user experience. Event Handling – managing events for adding, editing, and deleting tasks. Angular Material – integrating modern UI components for a more attractive and user-friendly interface. Angular Router – setting up navigation between components using routes and the RouterModule.
